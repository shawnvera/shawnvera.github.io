# Onboarding Sprint 2 blog post #

*What are you thoughts about the roles HTML, CSS, and JavaScript play in the process of rendering content and providing user experience?*

I feel that HTML and CSS are both familiar with a lot of people, but JavaScript is what really throws a curve ball at people. I've worked in all three languanges during the intro to web development class, and Javascript still has me a little confused. I can see that the 3 very different roles each language plays in a website may help to keep orthogonality in tact.

*What are you thoughts on pseudocoding?*

Pseudocoding is difficult for me as I like to just jump in an "get my hands dirty" so to speak. After this past week I can definitely see the value in pseudocode. It makes for a great starting point on a project, and helps to clear the logic out before coding.

*What was helpful during onboarding that made working remotely easier?*

The coaches availability helped a lot with this bootcamp being remote. They were always available to help whenver it was needed, which is much appreciated.

*What hindered your progress during onboarding?*

IF anything hindered my progress it may have been my need to figure things out on my own. I'm going to be working in the coming weeks on asking for help more frequently as I don't want to spend too much time with a blocker.

*How did you overcome challenges during onboarding?*

By asking questions and using the resources provided to find answers. I also used Google and Youtube quite often.
